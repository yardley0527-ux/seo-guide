🔗 Y-03｜Open Graph 標籤設定 最高優先

平台：Shopline ｜ 預計時間：20 分鐘 ｜ 難度：⭐⭐

為什麼要做？
Open Graph 決定網址被分享到 Facebook、LINE 時顯示的預覽圖、標題和描述。沒設定就只顯示空白。

1

先確認目前狀況

  1. 前往 Facebook 分享偵錯工具
  2. 輸入 https://www.yardley.tw/
  3. 點選「偵錯」
  4. 查看 og:title、og:description、og:image 是否有值
✅ 如果三個欄位都有值→ Shopline 已自動設定，跳到 Step 3 確認圖片
如果欄位是空的→ 繼續 Step 2
2

準備 OG 圖片

製作一張品牌主視覺圖，規格：

規格建議值
尺寸1200 x 630 像素
格式JPG 或 PNG
大小< 300KB
內容品牌 Logo + 主視覺 + 品牌名稱文字

在 Shopline 後台 → 「網路商店」→「偏好設定」 → 找到「社交分享圖片」上傳

3

手動加入 OG 標籤（如後台沒有此選項）

  1. 前往 「網路商店」→「主題」→「編輯 HTML/CSS」
  2. 找到 layout/theme.liquid
  3. 在 </head> 之前貼入：
<!-- Open Graph Tags -->
<meta property="og:type" content="website" />
<meta property="og:site_name" content="苼莛國際生技" />
<meta property="og:title" content="{{ page_title }} | 苼莛國際生技" />
<meta property="og:description" content="{% if page.description != blank %}{{ page.description | strip_html | truncatewords: 30 }}{% else %}苼莛國際生技提供天然安全的保健食品，融合全球頂尖技術與優質原料。{% endif %}" />
<meta property="og:url" content="{{ canonical_url }}" />
<meta property="og:image" content="https://www.yardley.tw/YOUR-BRAND-IMAGE.jpg" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
⚠️ 將 YOUR-BRAND-IMAGE.jpg 換成您實際上傳到 Shopline 的圖片路徑
✅ 驗證方法回到 Facebook 偵錯工具 → 點「再次抓取」→ 確認三個 og: 欄位都有正確值

  "contactPoint": {
    "@type": "ContactPoint",
